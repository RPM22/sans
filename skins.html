How the skin system works:

The sansundertale.cf skin system takes the contentents of a skin folder and loads them into the website. Each file in a skin folder has a name that matches the website elements.



How to create your own skins:

You can create your own skin folders by downloading the template zip file below, extracting and editing the contents, and uploading the folder (unzipped) to the settings menu.



download skintemplate.zip

Important information:

As of now, the skin engine is brand new. Because of this, the only supported image file type is PNG, and the only supported font file type is TTF. Other image and font format support will be added in the near future.


Our system loads all user files using HTML blob URLs, which are a secure way of using user files in websites, without uploading them to the internet. All files are kept on your device, and are not edited or altered in any way.
